# Task-manager-API
A simple REST API built with Node.js and Express.js to manage tasks using a CRUD framework (create, read, update, delete)


<br>

## Features
- Create tasks
- Get all tasks
- Update tasks
- Delete tasks

<br>

## Tech Stack
- Node.js
- Express

<br>

## Run Locally
```bash
npm install
npm run dev
